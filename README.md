# Odin Recipes

## Description
This project is part of The Odin Project curriculum. It is a simple recipe website built using basic HTML to practice structural tags, lists, links, and images.

## Skills Demonstrated
By completing this project, I will have demonstrated proficiency in:
* Setting up a local Git repository and linking it to GitHub via SSH.
* Creating and managing file structures.
* Using foundational HTML elements (headings, paragraphs, lists, and images).
* Utilizing Git workflows (staging, committing, and pushing changes).
